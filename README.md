<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Tracking Salary with Bifercation -(Additions and Deductions)
*   Tracking Expenses
*   Tracking Loans
*   Creating and managing budgets
*   Saving and viewing important documents
*   Generating detailed PDF reports
